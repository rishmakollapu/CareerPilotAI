# 🚀 CareerPilot AI

CareerPilot AI is an AI-powered career guidance platform that helps students discover career paths, identify skill gaps, receive personalized learning roadmaps, and analyze resumes.

Built for the **Ship to Get Hired – Gappy AI Hackathon 2026**.

---

## ✨ Features

- AI Career Guidance (Gemini AI)
- Career Readiness Score
- Personalized Learning Roadmap
- Recommended Skills, Courses & Projects
- AI Resume Analyzer
- Resume Skill Detection & Scoring
- Secure Login & Registration
- MongoDB Atlas Integration
- Responsive Dashboard

---

## 🚀 How It Works

1. Register/Login
2. Enter academic details and skills
3. Select a career goal
4. Receive AI-powered career analysis
5. Get personalized roadmap, courses, and projects
6. Upload a resume for skill extraction and scoring
